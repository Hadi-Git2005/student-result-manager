# Student Result Manager

This is a small Python project to manage student results.  
It allows you to store student details, calculate rankings, find toppers, and check weak subjects.

---

## Features Implemented (Day 2)

- Added **JSON-based data storage** so student records are saved persistently  
- Loaded student data from `students.json` instead of hardcoding  
- Created a **storage module** to handle reading and writing JSON  
- Added **dynamic student addition** via user input  
- Automatically saves newly added students to JSON  
- Implemented **basic input validation and error handling**  
- Rankings, toppers, and weak subjects now work with updated data
